Клиент-Серверное приложение для получения времени

Описание проекта
клиент-серверное приложение на Python. Сервер возвращает текущее время в формате JSON.

Структура проекта:
server/
├── server.py    - HTTP сервер
├── client.py    - Клиентское приложение
└── README.md    - Документация

Требования
- Python 3.6+
- Библиотека requests

Установка зависимостей
```bash
pip install requests
```

Запуск приложения

1. Запуск сервера
```bash
python server.py
```
Сервер: http://localhost:8000

2. Запуск клиента
В отдельном терминале:
```bash
python client.py
```



Используемые библиотеки
- http.server - для создания HTTP сервера
- requests - для HTTP запросов от клиента
- json - для работы с JSON форматом
- datetime -работыа с временем
